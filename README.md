# 🦀 Learning Rust

A progressive, hands-on journey through the Rust programming language — from first principles to advanced concepts. Every topic is coded from scratch, with a focus on understanding *why* things work the way they do, not just how to write the syntax.

Each topic includes working code, explanations of the reasoning behind language design decisions, and examples of both correct and incorrect patterns so you know what to avoid and why.

---

## Setup

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

cargo new my-project && cd my-project

cargo run      # compile and run
cargo check    # fast type-check
cargo fmt      # format
cargo clippy   # lint
cargo test     # run tests
```

**Editor:** VS Code + [rust-analyzer](https://marketplace.visualstudio.com/items?itemName=rust-lang.rust-analyzer)
